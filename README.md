**Un reloj mundial con estética retro de Game Boy.**  
Añade cualquier ciudad del mundo, busca con autocompletado, visualiza la hora en tiempo real con indicador día/noche y cambia entre modo claro y oscuro.  
Incluye sección "About Me" con enlaces a Discord y Ko-fi.

## ✨ Características

- 🕰️ **Hora en tiempo real** de cualquier capital o zona horaria IANA.
- 🌍 **Más de 150 ciudades** precargadas de todos los continentes.
- 🔍 **Buscador inteligente** con autocompletado: escribe "B" y sugiere "Bogotá", "Berlín", "Buenos Aires"...
- ☀️🌙 **Indicador visual de día/noche** junto al nombre de la ciudad.
- 🎨 **Modo oscuro / claro** al estilo Game Boy (verde clásico o verde oscuro).
- ❌ **Elimina ciudades** con un solo clic.
- 💾 **Guardado automático** en localStorage: tus ciudades persisten al cerrar el navegador.
- 👤 **Sección "About Me"** con enlaces a tu servidor de Discord y Ko-fi.
- 📱 **Totalmente responsive** para PC y móvil.
- ⚡ **Hecho con Luxon** para manejo preciso de zonas horarias.

---

## 🛠️ Tecnologías utilizadas

- **HTML5** – Estructura semántica.
- **CSS3** – Diseño retro pixelado, variables CSS, responsive design.
- **JavaScript (ES6+)** – Lógica de relojes, autocompletado, localStorage, eventos.
- **[Luxon](https://moment.github.io/luxon/)** – Biblioteca para manejo de fechas y zonas horarias.
- **[Google Fonts](https://fonts.google.com/)** – Fuentes *Press Start 2P* y *VT323* para la estética pixel.
